# thumbSwap

![](/presentation.gif)

This extension allows content creators to preview and test how video thumbnails and titles appear directly in the YouTube feed before publishing.

It works by temporarily replacing thumbnails and titles in real time on YouTube pages, enabling creators to simulate different presentation styles and improve their content performance.

### Privacy

No data is collected, stored, or transmitted. All changes are applied locally in the user’s browser session.

### IA usage

This extensions is almost completely vide-coded, I code it just for me cause no one did a easy extension like this, so I decide to push it so everyone can create it's own extenions on top of mine

### README (FR)

Cette extension permet aux créateurs de contenu de prévisualiser et tester l’apparence des miniatures et titres directement dans le fil YouTube avant la publication.

Elle fonctionne en remplaçant temporairement les miniatures et titres en temps réel sur les pages YouTube, afin de simuler différentes présentations et optimiser l’impact des vidéos.

### Confidentialité

Aucune donnée n’est collectée, stockée ou envoyée. Toutes les modifications sont effectuées localement dans le navigateur de l’utilisateur.
